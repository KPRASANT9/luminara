**Specific substrate**: `csos ring=$SUBJECT detail=cockpit`
Report: gradient, speed, rw, decision, F, mode, motor_entries.

**Everything**: Three calls:
  `csos ring=eco_organism detail=cockpit` → organism decision
  `csos action=greenhouse` → sessions, convergences
  `csos action=muscle` → motor memory top substrates

**F decomposition**: `csos equate=""` → vitality per ring, living equation state.

**Edge**: highest gradient + highest motor = best opportunity.

**Scale** ("check scale", "meta", "diminishing returns"):
  Read organism F trend: improving? stable? degrading?
  Read motor strength trends: rising or declining across substrates?
  Read greenhouse convergence stability: same pairs persisting or churning?

  Diminishing returns = organism_F > 0.40 AND top motors declining AND gradients flat.

  Root cause:
  - F rising across all → regime shift → re-run /globe + /rotate
  - F rising on specific subs → stale → heavy /feed with fresh data
  - F stable but gradients flat → redundant data → switch MCP source or timeframe

  Recurrence detection:
  - Read greenhouse for persistent convergences across timezone rings
  - Check motor intervals for periodic patterns (RhythmAtoms)
  - Cross-timezone lag: does Asia's state predict Europe? Track prediction accuracy.

  Report: overall yield (OPTIMAL/DEVELOPING/DIMINISHING), per-substrate health scores,
  recurrence patterns found, recommended adaptations.
