4. Boundaries

CanvasRead-only visualization, recommendations panel, SSE state displayNever executes commands, never accepts input, never calls interact
MembraneAll physics, recommend action, gradient, motor memory, Calvin atomsNever reasons about intent, never composes text

The Rule

Canvas OBSERVES. Agents ACT. Membrane COMPUTES.

Canvas surfaces what the membrane knows (recommendations). Agents collect what the membrane needs (question flow). Workflows define how substrates process (pipeline builder). Sessions are the living equations that run autonomously.
